# Support

## Getting Help

- Use GitHub Issues for reproducible bugs and feature requests.
- Read [README.md](README.md), [docs/conventions.md](docs/conventions.md), and [docs/official-creator-workflow.md](docs/official-creator-workflow.md) before opening a question.
- Use [SECURITY.md](SECURITY.md) for vulnerability reporting.

## What Maintainers Review

- Bugs that can be reproduced from the current `main` branch
- Documentation problems
- Reasonable feature proposals aligned with the project scope
- Contributions that keep the repo focused on skills, plugins, validation, and release readiness

## What Is Usually Out of Scope

- Private support via direct messages
- Debugging unrelated third-party setups
- Requests to maintain agent-specific formats that are not part of this repo
- Public security disclosure before a fix is available
